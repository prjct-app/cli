# /p:help

## Response

```
🎯 What do you want to do?

━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 🚀 Start working
   Say: "I want to start {task}"
   Or: /p:now "{task}"

2. ✅ Finish task
   Say: "I'm done"
   Or: /p:done

3. 🎉 Ship feature
   Say: "ship {feature}"
   Or: /p:ship "{feature}"

4. 💡 Capture idea
   Say: "idea about {thing}"
   Or: /p:idea "{idea}"

5. 📊 See progress
   Say: "show progress"
   Or: /p:recap

━━━━━━━━━━━━━━━━━━━━━━━━━━━

💬 Talk naturally - I understand both!
```
