# SOLUTION SECTION

## Headline
> ### One sync. Universal context.

## How it Works (3 steps)

**Step 1: Sync**
```bash
$ prjct sync
✓ Detected: TypeScript + React + Hono
✓ Generated: CLAUDE.md, CURSOR.md, GEMINI.md
✓ Extracted: 23 patterns, 4 conventions
```
> prjct analyzes your codebase and generates optimized context files for each AI agent.

**Step 2: Work**
```bash
$ prjct task "add user authentication"
→ Context injected automatically
→ Agent knows your stack, patterns, auth preferences
→ You ship. Fast.
```
> Your AI agent starts with full context. No explanations needed.

**Step 3: Learn**
```bash
$ prjct done
✓ Patterns extracted: 3 new
✓ Decisions logged: auth strategy, token handling
✓ Context updated for next session
```
> Every session makes the next one smarter.

## FAQ (Pricing)

**Q: Is the CLI open source?**
A: Yes! The core CLI is MIT licensed. Premium features require a subscription.

**Q: Can I self-host?**
A: The CLI runs locally. Team sync requires our cloud or your own infrastructure (Enterprise).

**Q: What counts as a "project"?**
A: Any directory with a `prjct.config.json`. Monorepos count as one project.
