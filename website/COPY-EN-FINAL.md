# WHAT IT DOES

## Three things

**Sync**
Analyze your codebase. Detect your stack. Generate optimized context files for each AI agent.

**Learn**
After a few tasks, prjct knows your branch naming, commit style, and library preferences. It applies them automatically.

**Focus**
Load only relevant files. Your agent sees 23 files, not 2,341. Less noise, better output.

# DAILY WORKFLOW

```bash
# Morning: sync your project
$ p. sync

# Start a task
$ p. task "add user authentication"
# → Context loaded: backend, database agents
# → Patterns applied: your JWT preference, your test style

# Work with Claude, Cursor, or Gemini
# They already know your stack

# Done
$ p. done
# → Duration tracked
# → Patterns updated

# What's next
$ p. next
# → Shows your task queue
```

**Is the CLI open source?**
Yes. MIT license. The core workflow runs locally on your machine.

**Can I try Pro before paying?**
