⚡ Quick Installation

📦 Option 1: One-Line Install (Recommended)

curl -fsSL https://jlopezlira.github.io/prjct-cli/install.sh | bash

Automatically installs and configures everything

🔧 Option 2: Clone from GitHub

git clone https://github.com/jlopezlira/prjct-cli.git
cd prjct-cli && ./setup.sh

📝 Available Commands

/p:init Initialize a new project
/p:now [task] Set or show current focus
/p:done Mark current task complete
/p:ship <feature> Ship and celebrate a feature 🎉
/p:idea <text> Capture ideas quickly
/p:recap Show project progress
/p:stuck <issue> Get contextual help
